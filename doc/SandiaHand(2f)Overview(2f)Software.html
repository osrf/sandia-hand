Software Access

The Sandia Hand software is being developed by the Open Source Robotics Foundation, and is completely open-source from top to bottom. The software is compiled using the conventional build tools (CMake, GNU Make, and GCC on Linux) and is provided as the sandia_hand_driver ROS package.
