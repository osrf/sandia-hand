Like the previous tutorial, this tutorial assumes that you have  sandia_hand_node  and  simple_grasp.py  up and running in their own terminals.

Now, get a new terminal and, as usual, source  ~/sandia-hand-ws/devel/setup.bash .

Then, 

rosrun sandia_hand_teleop asl_gui.py

This will pop up a gigantic text box. Type something in there (for example, "hi") and hit [ENTER]. The hand will slowly spell out the letters and then return to the home position.
